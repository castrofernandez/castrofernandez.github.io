# juancastro.es

Made with [Middleman](https://middlemanapp.com/)

1. Install

    * Middleman must be installed first.
    * Clone
    * Bundle install

2. Init project

```
middleman
```

3. Build project

```
middleman build
```

4. Git branches

    * dev: source code from Middleman's project.
    * master: output of Middleman's build command.
